El problema de la autopista: Un algoritmo para resolver un problema de grafos.

Dados n puntos en el plano, ¿Cuál es la mejor forma de elegir m conexiones entre ellos para minimizar la distancia promedio entre dos de ellos? Para responder a esta pregunta, hemos creado un maravilloso programa en Javascript que utiliza el algoritmo de Dikjistra y la triangulación de Delaunay para encontrar esa combinación de conexiones lo más rápido posible.
